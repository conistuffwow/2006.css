So, you want to use this library, eh?

Using 2006.css in your websites is very easy.

First, you must select a theme.

There are currently 4 themes available:

  • blue
  • orange
  • purple
  • red

Pick one you'd like, and then import it into your. HTML's "head" tag.


link rel="stylesheet" href="https://github.com/conistuffwow/2006.css/css/*insert theme here*.css"


you should see a fairly basic blank page. Don't worry, we'll fix this!
